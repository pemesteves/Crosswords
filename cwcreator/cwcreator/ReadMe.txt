Todos os objetivos foram cumpridos sendo que ambos os programas funcionam bem em Windows

cwcreator:

Quando o utilizador introduz uma posição, o programa funciona como case-insensitive, pois é chamada a função toupper para a linha e a função tolower para a coluna.
	- Facilita o tratamento da linha e da coluna, e mais facilidade para o utilizador, uma vez que este pode não usar minúsculas.

Quando é criado o dicionário, se o utilizador se enganar no nome do mesmo, o programa encerra.

Sempre que é perguntado ao utilizador se ele quer gravar o seu tabuleiro e este premir a tecla Y será perguntado se vai querer continuar depois ou terminar ali o desenvolvimento do seu projeto.
Se escolher terminar, as células com pontos ficarão com um #.

Foi adicionada a opção + quando se introduz a palavra: permite verificar se a palavra que se encontra a partir dessa posição é válida. 
A palavra é mostrada ao utilizador juntamente com uma frase que se diz se esta é válida ou não.
Ao mostrar a palavra pretende-se que o utilizador saiba se não se enganou na posição que queria inserir.
